# 希马交易所 - 全球高频极速安全的数字加密货币交易平台

这是一个基于 React 18 + Vite + Tailwind CSS 以及 Node 容器部署的希马交易所前端项目。

## 部署说明 (Hugging Face Spaces)

本项目已配置好完整的 Docker 镜像及 Nginx 转发规则，端口已统一定向为 **7860**。

如果您需要直接通过 GitHub 自动同步推送至 Hugging Face Space 运行，请参阅目录中的 `.github/workflows/sync-to-huggingface.yml`。
